p, a, h1~6, href, target

img 태그 src 연습

<p>태그는 paragraph, 즉 문단의 약자로, 하나의 문단을 만들 때 쓰입니다.

<a>태그는 하이퍼링크를 걸어주는 태그입니다.
      - href : 클릭시 이동 할 링크
      - target : 링크를 여는 방법

<img src="이미지경로" alt="">
      - src 특성은 필수이며, 포함하고자 하는 이미지로의 경로를 지정합니다.
         (./내 위치 ../상위 폴더 ../../두단계 위 폴더)

<img src="이미지경로" alt="">
      - alt의 3가지 용법 / 스크린리더기에게 그림을 이해할 수 있게 정보를 전달 / 검색엔진 최적화 / 이미지가 없을때 텍스트로 대체

<css>
* 전체 배경 색 *
       - body { background-color: ; }
* 태그 텍스트 색상 변경 *
       - h1 { color: 텍스트 색상; background-color: 텍스트 배경 색상; }
