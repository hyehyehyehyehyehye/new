★제목요소 h$*6
h1~h6 요소는 6단계의 구획 제목을 나타냅니다. 구획 단계는 h1이 가장 높고 h6은 가장 낮습니다.

★Entity Code
엔티티 코드란 Html 문서에서 특수 문자를 입력하기 위해 사용하는 코드 입니다.
W3C는 엔티티 코드가 아닌 특수 문자는 오류로 인식하기 때문에 특수 문자는 엔티티 코드로 작성하는 것이 좋습니다.
