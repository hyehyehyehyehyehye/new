💻텍스트 관련 요소

  1. em,i

    2. <em>태그는 강조된 텍스트 <i>요소는 이탤릭체로 표현

  2. strong,b

    3. <strong>은 중요한 텍스트 <b>다른 부가적인 목적 없이 단순히 굵게 표현

  3. address

    4. <address>
    요소의 텍스트는 주로 이탤릭체로 표현->문서나 글의 저자 또는 회사와 연락할 수 있는 정보를 명시

  4. mark,small

    5. <mark>는 하이라이트<small>은 텍스트 크기 작게

  5. sub,sup

    6. <sub>은 텍스트 아래로<sup>는 텍스트 위로
